# Olay × TikTok Shop — Operator Playbook

Internal strategy playbook for managing Olay on TikTok Shop.  
Built by Pattern Technologies — Social Commerce.

## Deploy to GitHub Pages

1. Create a new GitHub repo named `olay-tiktok-playbook`
2. Upload the contents of this folder
3. Go to **Settings → Pages → Branch: main → / (root)** → Save
4. Your playbook will be live at `https://[your-username].github.io/olay-tiktok-playbook/`
